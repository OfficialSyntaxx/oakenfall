Fact sheet

  • Title: Oakenfall
  • What it is: A dark-medieval isometric city-builder / settlement sim
  • Platform: Any modern browser — built touch-first for phones, plays equally on desktop
  • Price: Free
  • Install size: One HTML file. The entire game — every sprite, sound, and system — ships in a single self-contained file with zero runtime network requests
  • Play it: the gate is open

Features

  • 19 buildings with real supply chains (timber → planks, grain → flour → bread), livestock, decay, and repair
  • Villagers with names, trades, traits, families, morale — they court, marry, raise children, and walk out if you fail them
  • Four seasons and living weather; winter freezes the river solid
  • Rivers, fords, and bridges — every crossing is both an opportunity and a door for raiders
  • Wolves, bandits, palisades, guard posts, and a deep research tree
  • Four game modes from peaceful Settler to the merciless Iron Winter
  • Offline progress: close it mid-winter and the hold keeps working while you're away

Art direction

Old-school-MMORPG mood: earth-toned, painterly, candle-lit. Kenney block terrain and Tiny Swords characters, hand-graded darker to sit in a world of umber, moss, and ember. Day/night lighting with warm light pools; snow, rain, and drifting clouds.
